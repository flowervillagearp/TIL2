# more
리눅스에서 파일을 열때, 터미널 화면상에 다 보여지지 않을때 페이지 단위로 출력하는 명령

### 사용법
---
more [option] [filename1] [filename2]

### option
---
* more명령어로 페이지가 열렸을때
    1. 스페이스바+z: 다음 페이지로
    2. b: 이전페이지로
    3. enter: 한줄씩 넘김
    4. d: 반페이지씩 넘김
    5. f: 2페이지씩 넘김
    6. =: 현재 출 번호 출력
    7. :f: 현재 파일명과 줄 번호를 출력
    8. /[string]: 현재 파일에서 string값 찾아줌
    9. !: 다른 명령어 입력 가능 상태로 전환
    10. v: 현재 화면에서 vi편집창으로 전환
    11. h: 도움말
    12. q: 현재 명령 종료
